Programming Languages study

Its important to stay informed about trends in programming languages and technologies. Knowing what languages are growing or shrinking can help you decide where to invest. An excellent source to gain a better understanding of popular technologies is Stack Overflow. Stack overflow is an online question-and-answer site for coding topics. By looking at the number of questions about each technology, you can get an idea of how many people are using it.

In this project, I extract :

  1. The fraction of the total number of questions asked in 2019 had the tag R

  2. The five most asked tags over the last 5 years (2015-2020)

datasets/stack_overflow_data.csv

  • year: The year the question was asked.
  • tag: A word or phrase that describes the topic of the question.
  • number: The number of questions with a certain tag in that year.
  • year_total: The total number of questions asked in that year.

Fraction of the total number of questions asked in 2019 had the tag R

# ---------- 3. Q1 : r_percentage ----------
r_percentage <- df_stack_overflow %>%
    filter(year == 2019) %>%
    filter(tag == "r") %>%
    summarize(fraction = number/year_total*100)
print(r_percentage)
##    fraction
## 1 0.9656728

The fraction of the total number of questions asked in 2019 with the tag R is 0.9656728 .

Five most asked tags over the last 5 years (2015-2020)

highest_tags <- df_stack_overflow %>%
    filter(year >= 2015, year <= 2020) %>%
    group_by(tag) %>%
    summarise(n = sum(number)) %>%
    arrange(desc(n)) %>% slice(1:5) %>% pull(tag)

highest_tags <- as.character(highest_tags)
