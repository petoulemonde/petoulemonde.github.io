    Rise and Fall of programming languages

    The project

    It’s important to stay informed about trends in programming languages and technologies. Knowing what languages are growing or shrinking can help you decide where to invest. An excellent source to gain a better understanding of popular technologies is Stack Overflow. Stack overflow is an online question-and-answer site for coding topics. By looking at the number of questions about each technology, you can get an idea of how many people are using it.

    In this project, I extract :

    1. The fraction of the total number of questions asked in 2019 had the tag R

    2. The five most asked tags over the last 5 years (2015-2020)
